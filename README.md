2. **Commit to repository:**
   ```bash
   git add .
   git commit -m "Description of changes"
   git push origin rebuild-vanilla-js
   ```

3. **Deploy to AWS:**
   ```bash
   # Sync files to S3
   aws s3 sync "Marquel Yvette Photography/new-site/" s3://marquelyvette-website/ \
     --delete --exclude ".DS_Store" --exclude "*.md"

   # Invalidate CloudFront cache
   aws cloudfront create-invalidation \
     --distribution-id E50QXXWNUFNYT \
     --paths "/*"
   ```

4. **Verify deployment:**
   - Check CloudFront URL for immediate results
   - Check live site after cache invalidation completes (1-3 minutes)
